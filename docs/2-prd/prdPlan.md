- §N+1 对外 scope §8 IC 映射

## 14. 产品级 vs 技术级 边界判定自检（每写一段问一遍）

写完一段就自问 4 问题：

| 问题 | 产品级（留 PRD） | 技术级（迁 3-1） |
|---|---|---|
| 这段话能回答"这个能力做什么"？ | ✅ 留 | ❌ 迁 |
| 这段话描述"具体怎么实现"？ | ❌ 迁 | ✅ 迁 |
| 这段话含伪码 / YAML schema / 配置参数表？ | ❌ 迁 | ✅ 迁 |
| 这段话产品经理不懂也不影响决策？ | ❌ 迁 | ✅ 迁 |

**更简洁判定**：
- 「写给产品经理看」→ 产品级 → PRD
- 「写给程序员看」→ 技术级 → 3-1
