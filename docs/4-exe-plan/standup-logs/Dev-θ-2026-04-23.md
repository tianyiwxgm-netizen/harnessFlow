## 关键发现 / 修复

1. **Node 25 `--webstorage` 冲突** (WP01): Node v25.4.0 内置 localStorage stub shadow 掉 jsdom/happy-dom 真实实现。修复: `cross-env NODE_OPTIONS=--no-experimental-webstorage` 写入 npm scripts + vitest env=happy-dom。

2. **vue-router `<router-view>` 递归渲染陷阱** (WP02): 直接 `mount(MainLayout)` 会导致 MainLayout 内的 `<router-view>` 重新解析路由链顶端，递归渲染 MainLayout × 2。修复: 测试改 mount App.vue（`<router-view />` 作为根），让 router 正常驱动 depth=0/1。

3. **catch-all 先于 guard** (WP02): 未知 tab path（如 `/tabs/definitely_fake`）在 guard 跑之前就被 `/:pathMatch(.*)*` 匹配并 redirect，导致 guard 的"unknown tab warn"分支 dead code。接受该行为（catch-all 兜底够用），改测试断言只验 final path。
