## 关键决策与对策

1. **tech-stack 分歧**：arch.md §4 锁定"Vue 3 CDN + 零 npm"与 exe-plan §3 要求的 Vite + npm 生态冲突。决议以 exe-plan 为准（DoD §8 要求 vue-tsc + vitest + 392 TC + IndexedDB + vue-virtual-scroller，CDN 方案不可行）。**待主会话同步更新 arch.md §4。**

2. **tests 目录命名**：用户反馈 `app/l1_XX` 字面 ID 语义弱。Dev-θ 自己产出用语义命名（`backend/bff/` + `tests/bff/`），不碰既有 `app/l1_02/06/09`（CODE-OWNERSHIP-MATRIX）。

3. **Node 25 localStorage 冲突**：Node v25.4.0 内置 `--webstorage` 默认启用，它的 stub localStorage（无 clear/removeItem 方法）shadow 掉 jsdom/happy-dom 的真实实现。修复：`package.json` scripts 加 `cross-env NODE_OPTIONS=--no-experimental-webstorage`。同时 vitest env 换到 happy-dom（更稳）。

4. **BFF 依赖临时安装**：fastapi/uvicorn/httpx pip install 到共享 `.venv`。pyproject.toml dev-group 未动（遵 CODE-OWNERSHIP-MATRIX）。**待主会话合并 Dev-θ 时同步加入。**
